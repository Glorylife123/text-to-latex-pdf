# 面向动态图学习的交通状态预测方法

作者：张三，李四

日期：2026年5月

摘要：本文围绕动态图学习在城市交通状态预测中的应用展开研究。针对路网拓扑随时间变化、交通状态存在非线性传播以及多源数据融合困难等问题，本文整理了一种基于时空图神经网络的建模流程，并分析其在短时交通预测任务中的适用性。

关键词：动态图学习；交通预测；图神经网络；时空建模；多源数据

## 引言

城市交通系统具有复杂的时空相关性。传统时间序列模型通常难以刻画道路之间的拓扑依赖，而静态图模型又难以反映交通流随时间变化的传播关系。

## 方法

本文将道路路段表示为图中的节点，将路段连接关系和交通相关性表示为边。节点特征包括速度、流量和占有率。模型首先通过图卷积聚合邻域信息，再通过门控循环单元刻画时间依赖。

公式说明：预测误差 E 等于 y_hat 减去 y。
