# 课程调研报告：生成式人工智能在科研写作中的应用

作者：李四

摘要：本文调研生成式人工智能工具在科研写作流程中的应用，包括文献检索、提纲生成、语言润色和格式转换。

关键词：生成式人工智能；科研写作；学术规范；工具使用

## 引言

生成式人工智能能够提升文本处理效率，但也带来事实核验、引用规范和学术诚信方面的问题。

## 调研发现

| 应用环节 | 典型任务 | 风险 |
| --- | --- | --- |
| 文献检索 | 主题扩展、关键词推荐 | 编造文献 |
| 写作辅助 | 提纲、摘要、改写 | 观点同质化 |
| 排版转换 | Markdown 转 LaTeX | 编译错误 |

## 公式说明

若用 E 表示效率提升比例，可定义为使用工具后的单位时间产出 P_ai 与人工基线 P_base 的相对增量。
