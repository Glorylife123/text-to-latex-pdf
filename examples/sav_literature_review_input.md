# SAV 文献精读报告：Attention Is All You Need

作者：王五

## 文献信息

Vaswani, A. et al. Attention Is All You Need. NeurIPS, 2017.

## 研究问题

论文希望在不使用循环神经网络和卷积网络的情况下，构建高效的序列到序列建模框架。

## 核心方法

Transformer 使用自注意力机制刻画序列内部依赖，并通过多头注意力捕捉不同表示子空间中的关系。

公式说明：Attention(Q,K,V) 等于 softmax(QK 转置除以 sqrt(d_k)) 乘以 V。

## 主要贡献

| 方面 | 内容 |
| --- | --- |
| 架构 | 完全基于注意力机制 |
| 效率 | 支持更高并行度 |
| 效果 | 在机器翻译任务上取得优秀结果 |

## 批判性思考

论文证明了注意力机制的表达能力，但对长序列的计算复杂度仍然较高。
